Camel Router Spring Project
===========================

To build this project use

    mvn install

To run this project with Maven use

    mvn camel:run

For more help see the Apache Camel documentation

    http://camel.apache.org/


Camel Spring Maven Project Generation
=====================================
 mvn archetype:generate -DarchetypeGroupId=org.apache.camel.archetypes -DarchetypeArtifactId=camel-archetype-spring -DarchetypeVersion=2.13.0 -DgroupId=com.indigoarc -DartifactId=kporg

 
 eclipse project generation
 mvn eclipse:eclipse
